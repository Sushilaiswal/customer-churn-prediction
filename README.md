# Customer Churn Prediction (Banking)

## Project Overview
This project predicts whether a customer will leave the bank using machine learning.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Model Used
- Logistic Regression / Random Forest

## Dataset
The dataset (`customer_data.csv`) consists of bank customer details such as demographic information, account activity, and financial attributes, which are used to build a machine learning model for churn prediction.

## How to Run
1. Install requirements
2. Run the notebook or Visual Studio

## Conclusion
The project demonstrates how machine learning can be used to predict customer churn and assist banks in improving customer retention.
